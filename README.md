### docure-header

```js
npm i @specialdoom/docure-header
```

### usage

```html
<docure-header>
  <docure-header-item>
    <a href="#">Nav item</a>
  </docure-header-item>
</docure-header>
```

#### docure-header-item
 - _isright_ attribute to float the item to right
